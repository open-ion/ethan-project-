# イオン専用 確定プラン（パラメータ確定版）

> 2026-06-21 ヒアリング確定。毎月の余力 **3〜7万円** ／ 目標ペース **20年で1800万（標準・推奨）**。
> ※生活防衛資金（生活費6か月分）の有無は未確認。**もし未確保なら、それを先に普通預金で埋めるのが全ての前提。**

## 1. 毎月いくら？ → 「5万円から始めて、7万円に伸ばす」

- 目標「20年で生涯枠1800万」を**枠ぴったり完走**するには月7.5万必要。君の余力上限は7万なので、**月7万ならほぼ達成（21年で完走）**。
- いきなり7万は重い。**まず月5万でスタート → 固定費削減（保険見直し・サブスク整理）で月7万へ寄せる**のが現実的。藤原氏も「超余裕資金は作り出すもの」と言っている。新しく稼ぐ前に、まず固定費から絞り出す。

### 20年後の到達点（つみたて継続・税引前のざっくり試算）
| 毎月 | 投資元本(20年) | 3%想定 | 5%想定 | 7%想定 |
|---|---|---|---|---|
| 3万 | 720万 | 985万 | 1,233万 | 1,563万 |
| **5万（開始）** | 1,200万 | 1,642万 | **2,055万** | 2,605万 |
| **7万（目標）** | 1,680万 | 2,298万 | **2,877万** | 3,646万 |
| 7.5万（枠完走） | 1,800万 | 2,462万 | 3,083万 | 3,907万 |

> 「10%」は本のタイトルの理想値。**現実的に効くのは5%前後**。NISAなので**この増えた分は全額非課税**。これが最大の武器。

## 2. 成長投資枠、いつから？ → 「条件を満たせば"今"から。ただし個別株は貯まってから」

判定（上から順に）：
1. **生活防衛資金OK？**（生活費6か月〜1年が普通預金に）→ 未確保ならまずここ。
2. **オルカンのつみたてが家計を圧迫せず回ってる？** → YESで次へ。
3. → **その上の超余裕資金は、出た分だけ成長投資枠OK。**

具体トリガー：
- 投信（オルカン買い増し・リートインデックス）は**100円から成長投資枠で即OK**。
- **個別株は「サテライト用の積立」が貯まってから**：
  - 大江戸温泉リート … **約5.4万円**貯まったら1口
  - 三菱HCキャピタル … **約8万円**貯まったら1単元（S株/かぶミニで月々コツコツでも可）
  - イオン株（優待） … 100株で約100万円。**当面は優先度を下げる**（まとまるまで時間がかかるため、後回しでよい）

## 3. 毎月5万円の配り方（フェーズ別）

### フェーズ1：最初の12か月（慣れる期）
```
オルカン（つみたて投資枠）  5万円  ← 全額これ一本
```
1年は迷わずオルカン1本。値動きに慣れ、ほったらかしの感覚を掴む。

### フェーズ2：13〜24か月（サテライト解禁・余力7万に増やせていれば）
```
オルカン（つみたて枠）        4.0万円
高配当株 積立（成長枠）        2.0万円  ← 三菱HCキャピタル/KDDI/三井物産を単元未満で
REIT or リートインデックス     1.0万円
```

### フェーズ3：25か月〜（完成形に寄せる／月7万前提）
```
オルカン（コア）              4.0万円  (57%)
高配当株（個別・成長枠）       1.5万円  (21%)
REIT                          0.7万円  (10%)
優待株（イオン株へ積立）       0.5万円  ( 7%)
クラファン（特定口座・余剰）   0.3万円  ( 5%)
```
→ 目標配分 オルカン60/高配当株20/REIT10/優待5/クラファン5 に自然に着地。

## 4. 7月にやること（スタート手順）
1. （未了なら）生活防衛資金を普通預金で確保。
2. 証券口座でNISAの**つみたて投資枠**を設定し、**eMAXIS Slim 全世界株式（オルカン）を月5万円**で自動積立。
3. 固定費を1つ見直す（保険 or サブスク）。浮いた分を翌月の積立上乗せ候補に。
4. 「サテライト用」の口座/枠を別に意識して、超余裕資金をプール開始。
5. あとは**ほったらかし。月1回の積立確認だけ。**

## 運用の鉄則（再掲）
超余裕資金だけ／値動きを毎日見ない／**暴落で売らない・やめない**（相場に居続ける）／個別株は利回り3%超・配当性向50%以下・減配なしの間は持つ／高すぎる利回りは裏を疑う。
