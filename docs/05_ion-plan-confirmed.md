# イオン専用 確定プラン（キャッシュフロー反映・確定版 v2）

> 2026-06-21 実際のお金の流れを反映して全面改訂。

## 0. 現状の整理（君のお金の流れ）

| 収入源 | 月額 | 用途 |
|---|---|---|
| 本業＋アルバイト | 30〜35万 | 車ローン・奨学金・家賃・光熱費・生活費（**ほぼ全額消費。余剰ゼロ前提**） |
| 副業（6月開始） | 15万 | 税金3.5万／**現金バッファ7万**／**投資準備3.5万** |

- **防衛資金・貯金はまだほぼゼロ**（副業を6月に始めたばかり）。
- **実際の投資余力 ＝ 月3.5万**（投資準備ぶん）。
- 👏 税金3.5万を最初から分けてるのは正解。副業は確定申告が要る（雑所得/事業所得）。この習慣は崩さないこと。

### 重要な気づき：投資準備3.5万は、すでに「超余裕資金」の条件を満たしている
君の予算設計上、3.5万は「税金・バッファ・生活費を全部引いた後の、完全に余ったお金」。本業が生活費を賄っている以上、**この3.5万は無くても生活が回る ＝ 藤原の言う超余裕資金そのもの**。だから7月スタートはOK。ただし下の前提を守る。

## 1. 今は「二段ロケットの1段目 ＝ 防衛資金づくり」フェーズ

現金バッファ7万/月は、君の**生活防衛資金を作る正しい仕組み**。これを最優先で積む。

| 防衛資金の目標 | 必要額 | 7万/月で到達 |
|---|---|---|
| 最低ライン（生活費3か月） | 約100万 | **約14か月** |
| 安心ライン（生活費6か月） | 約180万 | **約26か月（2.1年）** |

> 副業は始めたばかりで収入が安定するか未知数。だからこそ防衛資金が先。**もし副業収入が落ちたり、車の故障など急な出費が来たら、迷わず投資準備3.5万を止めてバッファに回す。** これが暴落時に投資を売らずに済む保険になる。

## 2. 7月からのアクション（投資準備3.5万の使い方）

### フェーズA：今〜防衛資金が100万に届くまで（最初の約14か月）
```
現金バッファ        7.0万/月  → 防衛資金（最優先）
投資準備 オルカン    3.5万/月  → eMAXIS Slim 全世界株式（つみたて投資枠）一本
税金                3.5万/月  → 別口座でキープ
```
- **投資は全額オルカン。個別株はまだやらない。**（防衛資金が薄い間は、流動性が高く分散の効いたオルカンだけ。個別株は値動きが荒く、いざという時に売りたくなる）
- 目的は「習慣を作る・値動きに慣れる」。3.5万でも複利は20年で効く（下表）。

### フェーズB：防衛資金が満タン（約2年後）→ ここから本番、月10.5万に加速 🚀
防衛資金が6か月分(180万)に届いたら、**現金バッファに回していた7万が"空く"**。その7万を投資に合流：
```
投資準備 3.5万 ＋ 元バッファ 7万 ＝ 月10.5万を投資へ
```
- ここで初めて**成長投資枠で個別株を解禁**（防衛資金が100万を超えてから、が解禁ライン）。
- 配分は `docs/02` の完成形へ：オルカン60／高配当株20／REIT10／優待5／クラファン5。
- 第一候補：三菱HCキャピタル・KDDI・三井物産（利回り3%超×配当性向50%以下×減配なし）。

## 3. 20年の見通し（5%想定・税引前のざっくり）

| シナリオ | 20年後の資産 | 1800万枠の完走 |
|---|---|---|
| もし3.5万のまま20年 | 約1,440万（元本840万） | 43年（届かない） |
| **二段ロケット（2年3.5万→以後10.5万）** | **約3,880万** | **約15.6年で完走** |

> ポイントは「**今の3.5万に絶望しないこと**」。防衛資金が空ける7万が合流した瞬間、投資力は3倍になる。最初の2年は助走。そこを耐えれば、20年で1800万枠を完走して資産は3,000万超が見える。「10%」は理想値、現実は5%前後で十分これだけ届く。NISAだから増えた分は非課税。

## 4. 7月にやることリスト
1. 証券口座（SBI／楽天など）でNISAの**つみたて投資枠**を設定。
2. **eMAXIS Slim 全世界株式（オルカン）を月3.5万円**で自動積立。日付は副業入金後に。
3. 現金バッファ7万は引き続き別口座へ（防衛資金100万→180万を目標に）。
4. 税金3.5万も別口座で隔離キープ（確定申告に備える）。
5. あとは**ほったらかし。月1回、積立されたか確認するだけ。**
6. 防衛資金が100万を超えたら、イーサンに声をかけて「個別株フェーズB」へ移行。

## 運用の鉄則
超余裕資金だけ（＝投資準備3.5万の範囲）／値動きを毎日見ない／**暴落で売らない・やめない**／個別株はフェーズBから／高すぎる利回りは裏を疑う／**副業が不安定になったら投資を止めてバッファ優先**。
