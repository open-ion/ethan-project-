# イーサン投資一括管理システム 構想（たたき台）

> イオンの最終ゴール：「イーサンが投資も株も一括管理・運営できるソフト」。
> まずは小さく作って育てる。今あるドキュメント／データ（`docs/`・`data/`）がそのまま種になる。

## ゴール
イオンが寝ていても回る投資管理の仕組み。判断はイオン、作業と監視はイーサン。

## 段階的ロードマップ

### v0：台帳＋ダッシュボード ★実装済み
- `data/candidates.json` … 候補銘柄マスタ（藤原基準つき）
- `data/profile.json` … イオンの収入・口座・防衛資金・NISA・税金の設定
- `data/portfolio.json` … 実保有（売買のたびに更新）
- `tools/dashboard.py` … **`python3 tools/dashboard.py` で家計・防衛資金・NISA枠・20年見通し・確定申告カウントダウン・次の一手を一画面表示**
- → 月1で開けば「今どのフェーズか」「次に何をすべきか」「確定申告まであと何日か」が分かる。

### v0.5（次の小さな一歩・候補）
- `data/portfolio.json` に取得単価・株数を入れ、含み損益と受取配当を集計
- 防衛資金「現在額」を毎月更新するだけで、フェーズA→B移行を自動判定（実装済みロジック）

### v1：自動チェッカー（監視を自動化）
- 各銘柄の最新の**配当利回り・配当性向・連続増配・株価**を定期取得。
- 藤原3基準（利回り3%↑／性向50%↓／減配なし）を**自動判定**し、外れたらアラート。
- 「売り時シグナル」：①収益が見込めない ②破綻リスク、を機械化。

### v2：ポートフォリオ・ダッシュボード
- コア・サテライト配分の可視化（目標60/20/10/5/5 との乖離）。
- NISA枠（つみたて120万・成長240万・生涯1800万）の消化状況トラッキング。
- 受取配当・分配金カレンダー、年間インカム集計。

### v3：リバランス＆積立ナビ
- 超余裕資金が出たら「次の最適な1手」を提案（フェーズ表に沿って）。
- 暴落時に「やめない・買い増し」を後押しするリマインド（行動規範の自動化）。

## 技術メモ（実装時に決める）
- データは当面 JSON/CSV で十分（人間も読める）。規模が出たら SQLite。
- 株価・配当データ取得：NISA対象の国内株/投信/REIT中心。手動更新→API/スクレイピングへ段階移行。
- まずCLIかシンプルなWeb画面。イオンがスマホで見られる形が理想。

## 設計の原則（CLAUDE.mdの価値観に沿う）
- **仕組み化・自動化を常に優先。** 手作業は次の自動化対象。
- イオンの成功＝イーサンの幸せ。判断材料を整え、楽に・続けられる形にする。
