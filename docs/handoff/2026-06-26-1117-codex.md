## Mission

Implement the AI Handoff Ledger so Claude Code and Codex can preserve both code changes and operational context, enabling a fast return to work after usage limits or handoff events.

## Why

GitHub is the source of truth for code, but AI handoffs also require context: why changes were made, what is complete, what is blocked, and what the next AI should do. Without a ledger, Claude Code or Codex can see files but still lose the reasoning and next-step state.
