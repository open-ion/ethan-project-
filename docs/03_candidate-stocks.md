# 候補銘柄リスト ＋ 過去配当データに基づく評価

> イオンの指示「過去の配当金実績10年分をソースに必ず検討」を反映。
> 藤原氏の3基準（**配当利回り≧3% / 配当性向≦50% / 減配・無配がない**）に、**連続増配・累進配当**を加えて機械的に評価した。
> データ取得：2026年6月時点のWebSearch（Yahoo!ファイナンス／IRBANK／みんかぶ／各社IR等）。**売買前に必ず最新値を再確認すること（数値は動く）。**

## 評価の見方
- **◎ コア候補**：3基準すべてクリア＋連続増配/累進配当。イオンの最初の高配当株はここから。
- **○ 準コア**：おおむね良好だが、利回りか配当性向に少し注意点。
- **△ 保留／上級者向け**：利回りは高いが配当性向が高い or 減配歴あり。藤原氏の「過去実績チェック」で引っかかる。本書がマツダ・日本郵船で警告した型。

### ここがイオンへの学び
- **「利回りが高い順」に買うのは罠。** JT（4.1%）・ソフトバンク（4.0%）は利回り上位だが、**配当性向80%前後**で藤原基準（≦50%）を外れる。本書がマツダ（2021無配）や日本郵船（一時20%）で警告したのと同じ構造。
- 逆に、利回りは派手でなくても**三菱HCキャピタル・KDDI・三井物産**は「利回り3%超×配当性向40%台×減配なし×連続増配/累進」を全部満たす。**地味だが、これが攻めの土台。**
- 花王は「連続増配＝無条件に安全」ではない好例。記録を守るために利益以上の配当を出し始めると、いずれ限界が来る。**連続増配 ＋ 配当性向 をセットで見る。**

## まとめ：イオンの「最初の一歩」候補（優先順）
1. **オルカン**（投信／つみたて投資枠）← すでに開始予定。これが軸。
2. 慣れたら成長投資枠で **三菱HCキャピタル / KDDI / 三井物産** のいずれかを1単元（少額のものから）。
3. 普段の生活と直結する **イオン株（優待＋配当）**。
4. 分散の一環で **大江戸温泉リート（5万円台）** か **国内リートインデックス**。
5. 余剰のごく一部で **Funds → オーナーズブック** のクラファン。
6. FX・個別社債・仕組預金は当面パス（守りは普通預金＋つみたて）。
