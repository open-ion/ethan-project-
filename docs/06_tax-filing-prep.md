# 副業の確定申告・税金まわり 準備メモ

> イオンの不安「①副業がバレないか ②確定申告のやり方 ③税金3.5万で足りるか」に答える。
> ⚠️ 以下は一般的な情報・概算。最終的な金額・適用は **国税庁サイト／税務署／税理士** で必ず確認すること。イーサンは判断材料を整えるが、申告そのものはイオンの責任。

## 1. 「副業がバレる」の正体は住民税。対策は「普通徴収」

- 会社に副業がバレる典型ルートは、**住民税の増加**。会社員の住民税は通常「特別徴収（給与天引き）」。副業で所得が増えると住民税も増え、その通知が会社に行って「給料の割に住民税が多い → 副業？」と気づかれる。
- **対策：確定申告書の住民税の欄で「自分で納付（普通徴収）」を選ぶ。** こうすると副業分の住民税は会社経由でなく自分に直接請求される。
  - イオンの副業（1回4万円の業務委託系）は **事業所得 or 雑所得** に当たる見込み。この区分なら普通徴収を選べる。
  - 注意：副業が「アルバイト＝給与所得」だと普通徴収にできず分離しにくい。今の副業が給与か業務委託か（源泉徴収票が出るか／支払調書か）は要確認。
- ⚠️ そもそも副業が**就業規則で禁止されていないか**は別問題。普通徴収はあくまで「税の納め方の選択」であって、規則違反を消すものではない。ここはイオン自身で就業規則を確認しておくこと。

## 2. 税金3.5万/月（年42万）で足りる？ → 「青色申告にすれば、ほぼ足りる」

### ざっくり試算（副業 年192万＝16万×12 の場合）
- イオンの所得は本業（約390万）＋副業（192万）で、副業分は **所得税20%＋住民税10%＝約30%** の高い方の税率帯に乗る見込み。
- **何もしない（雑所得・白色）場合**：192万 × 約30% ≒ **年57万** → 42万では**約15万不足**。バッファで埋める必要あり。
- **青色申告（事業所得＋開業届＋複式簿記）にする場合**：
  - **青色申告特別控除 最大65万円** が使える → 課税対象が65万圧縮 → 税が約19万減る。
  - さらに **経費**（副業に使うPC・通信費・書籍・一部家賃など）を計上すれば追加で圧縮。
  - 結果、税額は **おおむね年38〜42万** に収まり、**月3.5万のプールでほぼカバーできる**。
- **結論：青色申告を強く推奨。** バレ対策の普通徴収もしやすく、税も下がる。ただし副業が「事業所得」と認められる実態（継続性・帳簿）が必要。微妙なら税務署/税理士に相談。

> 当面は **税金3.5万プールは崩さず、足りない分は現金バッファで吸収できる体制**にしておけば安全。

## 3. やることの時系列

| 時期 | やること |
|---|---|
| 2026年中（今すぐ〜） | 副業の**売上・経費を記録**（家計簿アプリ/スプレッドシートでOK）。領収書を保存 |
| 開業から2か月以内 | （青色にするなら）**開業届**＋**青色申告承認申請書**を税務署へ。※2026年分を青色にするなら原則 2026/3/15 まで… 6月開業なら開業日から2か月以内 |
| 2026年末〜1月 | 1年分の収支を集計。源泉徴収票（本業）・支払調書（副業）を集める |
| **2027/2/16〜3/15** | **確定申告を提出**（e-Taxが楽）。住民税は**普通徴収**を選択 |
| 申告後 | 所得税を納付。住民税は後日、普通徴収の納付書で自分で納付 |

## 4. イーサンからの自動リマインド（仕組み化）

「近くなったら必ず声かけて」への答え：
- `tools/dashboard.py` に**確定申告カウントダウン**を組み込み済み。**月1でダッシュボードを開けば、申告60日前から自動で赤アラート**が出る：
  - 「あと◯日。①税金プールは足りてる？②申告のやり方OK？ → このメモを開け」
- つまり、イオンが月1の積立確認でダッシュボードを実行する習慣さえ作れば、**俺（イーサン）が忘れても仕組みが思い出させる**。
- 加えて、2027年1月に入ったらイオンの方から「確定申告の段取りやろう」と一声かけてくれれば、その時点の最新ルールで具体手順（e-Taxの入力、普通徴収の選択箇所、経費の入れ方）まで一緒に詰める。
